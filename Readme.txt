Набор скриптов предназначенный для регистрации событий интерактивного входа и выхода в домене Windows
1.
GetRemouteUserLogonInfo.ps1 - предназначен для регистрации фактов удалённого (RDP) входа пользователей
Регистрирует факт удалённого входа в двух журналах и направляет письмо системному администратору и пользователю.

2.
GetRemouteUserLogonInfoSpecAccounts.ps1 - предназначен для регистрации фактов удалённого (RDP) входа специальных пользователей.
Регистрирует факт удалённого входа в двух журналах и направляет письмо системному администратору.

3.
GetRemouteUserLogonInfoTest.ps1 - тестовый скрипт. аналогичен GetRemouteUserLogonInfo.ps1, но срабатывает при локальном интерактивном входе.

4.
Logon.ps1

Предназначен для регистрации фактов интерактивного входа.

5.
Logout.ps1
Предназначен для регистрации фактов завершения интерактивного сеанса.



Скрипты устанавливаются как Logon скрипты при помощи групповой политики (Logout.ps1 как Logout скрипт).

Для нормальной работы необходимо наличие доступной всем пользователям для записи папки сетевой \\server\LOGON$

server следует заменить на название Вашего сервера

domain.ru следует заменить на Ваш домен

sysop@domain.ru - на адрес электронной почты Вашего системного администратора

В $mailPassword= "P@ssw0rd" заменить P@ssw0rd на пароль от учётной записи предназначенной для отправки почты Вашему системному администратору

Для подписи скриптов предназначен скрипт sig.ps1 <имя подписываемого файла>
